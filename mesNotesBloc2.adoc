= POO (Programmation Orientée Objet)
Une *CLASS* est un ensemble d'attributs et de méthodes (portée(type) un nom).

* Portée = public/protected/private

* Méthodes = fonctions liées à la class 

1. Constructeur, Afficheur, Acceseurs

2. Méthodes définis et conçus par le développeur

* Signature
1. Portée              
2. Mot clef function
3. Nom de la fonction
4. Paramètres

== Instanciation d'une classe
